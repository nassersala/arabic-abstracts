### English Version

The appendix discusses the relationship between Turing's notion of computability and Church's concept of λ-definability.

**Church's λ-calculus:** Alonzo Church independently developed a formalism (the λ-calculus) for defining computable functions. A function is λ-definable if it can be expressed in Church's system.

**Equivalence theorem:** Turing proves that a function is λ-definable if and only if it is computable by a Turing machine. This remarkable equivalence between two completely different formalisms strongly suggests both have captured the true nature of effective computability.

**Multiple equivalent formalisms:** Beyond Turing machines and λ-calculus, several other formalisms have been proposed:
- General recursive functions (Kleene)
- Combinatory logic (Schönfinkel, Curry)
- Random access machines
- Markov algorithms

All of these define exactly the same class of computable functions, further validating the Church-Turing Thesis.

**Philosophical significance:** The convergence of multiple independent formalisms on the same class of functions provides strong evidence that this class represents an objective mathematical concept, not an arbitrary choice of formalism.

**Conclusion:** Whether we use Turing machines, λ-calculus, or other equivalent systems, we arrive at the same fundamental limitations on computation. This universality underpins the entire field of computability theory.

**شكليات متكافئة متعددة:** بالإضافة إلى آلات تورينغ وحساب لامدا، تم اقتراح عدة شكليات أخرى:

**الخلاصة:** سواء استخدمنا آلات تورينغ أو حساب لامدا أو أنظمة متكافئة أخرى، نصل إلى نفس القيود الأساسية على الحوسبة. هذه العالمية تدعم مجال نظرية قابلية الحوسبة بأكمله.
