### English Version

A protocol that supports the sharing of resources that exist in different packet switching networks is presented. The protocol provides for variation in individual network packet sizes, transmission failures, sequencing, flow control, end-to-end error checking, and the creation and destruction of logical process-to-process connections. Some implementation issues are considered, and problems such as internetwork routing, accounting, and timeouts are exposed. This paper described the Transmission Control Protocol (TCP), which later split into TCP and Internet Protocol (IP), becoming core components of the Internet.
