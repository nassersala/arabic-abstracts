## Paper Overview

This is a tutorial paper introducing Remora, a higher-order, rank-polymorphic array-processing programming language designed for parallel hardware execution. The paper covers:

1. Dynamic Remora (with dynamic typing)
2. Explicitly Typed Remora (with static dependent type system)
3. The rank-polymorphic computational model
4. Core mechanisms: frame polymorphism, principal-frame cell replication, and reranking

The paper serves as both an introduction to Remora and to the general rank-polymorphic array-processing computational model pioneered by Iverson for APL.
