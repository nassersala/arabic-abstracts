# Learn Physics by Programming in Haskell
## تعلم الفيزياء من خلال البرمجة في Haskell

**arXiv ID:** 1412.4880
**Authors:** Scott N. Walck
**Year:** 2014
**Categories:** cs.CY, cs.PL, physics.ed-ph
**Translation Quality:** 0.87
**Glossary Terms Used:** functional programming, programming language, higher-order, type system, function

### English Abstract
The paper describes "a method for deepening a student's understanding of basic physics by asking the student to express physical ideas in a functional programming language." The approach was implemented in a second-year computational physics course. The authors argue that "the structure of Newtonian mechanics is clarified by its expression in a language (Haskell) that supports higher-order functions, types, and type classes." They demonstrate how type signatures in electromagnetic theory effectively capture "the functional dependency on the charge and current distributions that produce the fields." The work illustrates how "many of the ideas in basic physics are well-captured by a type or a function."
