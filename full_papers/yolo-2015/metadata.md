## Paper Summary

YOLO (You Only Look Once) is a groundbreaking object detection system that frames detection as a regression problem rather than a classification problem. It uses a single neural network to predict bounding boxes and class probabilities directly from full images in one evaluation, making it extremely fast (45 fps) while maintaining good accuracy. This paper introduced the foundational YOLO architecture that spawned numerous iterations (YOLOv2-YOLOv11+).
