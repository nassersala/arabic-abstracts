## Paper Overview

This paper presents SLICED (State Linked Interface Compliance Engine for Data), a methodology for generating formal specifications from conventional engineering models. The approach is demonstrated using NASA's Virtual ADAPT model and validated with Simulink simulation.

**Key Contributions:**
- Compositional approach to formal state machine specification
- Methodology for generating FSM from conventional engineering models (AADL, SysML, Simulink)
- Subsystem merging techniques for performance optimization
- Case study using NASA Virtual ADAPT (electrical power system)
- Validation of error detection via Simulink simulation
