### English Version

We demonstrate how methods in Functional Programming can be used to implement a computer algebra system. As a proof-of-concept, we present the computational-algebra package. It is a computer algebra system implemented as an embedded domain-specific language in Haskell, a purely functional programming language. Utilising methods in functional programming and prominent features of Haskell, this library achieves safety, composability, and correctness at the same time. To demonstrate the advantages of our approach, we have implemented advanced Gröbner basis algorithms, such as Faugère's F₄ and F₅, in a composable way.

**Keywords:** Gröbner basis; signature-based algorithms; computational algebra; functional programming; Haskell; type system; formal methods; property-based testing; implementation report.
