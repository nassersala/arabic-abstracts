## Paper Summary

This paper reports on an experimental case study by ESG (a German medium-sized company) examining the application of formal verification to safety-critical avionics software development. The study formalized low-level requirements using ACSL (ANSI C Specification Language) and verified source code using the Frama-C framework with the WP plugin and Alt-Ergo automated prover. The paper establishes fitness criteria for industrial use in small and medium enterprises (SMEs) and identifies challenges specific to embedded real-time programming in the avionics domain, particularly in the context of DO-178C certification guidelines.
