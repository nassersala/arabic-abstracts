## Paper Summary

This paper studies formal utility and privacy guarantees for random matrix compression of databases. The key contribution is showing that differential privacy can be achieved while preserving utility for statistical learning tasks like Principal Component Analysis (PCA) and sparse regression. The approach uses random Gaussian projections to compress data, reducing the number of records while maintaining statistical utility.
