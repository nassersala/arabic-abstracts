# Project Elements: A computational entity-component-system in a scene-graph pythonic framework, for a neural, geometric computer graphics curriculum
## مشروع Elements: نظام حوسبي للكيان-المكون-النظام في إطار عمل بايثوني للرسم البياني للمشهد، لمنهج دراسي في الرسومات الحاسوبية العصبية والهندسية

### English Abstract
The Elements project is a lightweight, open-source, computational science and computer graphics framework, tailored for educational needs that combines Entity-Component-System architecture with scene-graph capabilities using Python. The framework aims to teach computer graphics concepts—including animation, skinning, geometric algebra, and graph neural networks—in a transparent, non-black-box manner.
