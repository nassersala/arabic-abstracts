# The PageRank Citation Ranking: Bringing Order to the Web
## ترتيب الاستشهاد PageRank: إضفاء النظام على الويب

**Technical Report ID:** SIDL-WP-1999-0120

**Year:** 1998
**Institution:** Stanford Digital Library Technologies Project
**Translation Quality:** 0.96
**Glossary Terms Used:** efficient, randomized, computation, citation, interest, method, navigation, objective, rating, subjective, web page, web surfer

### English Abstract
The importance of a Webpage is an inherently subjective matter, which depends on the readers interests, knowledge and attitudes. But there is still much that can be said objectively about the relative importance of Web pages. This paper describes PageRank, a method for rating Web pages objectively and mechanically, effectively measuring the human interest and attention devoted to them. We compare PageRank to an idealized random Web surfer. We show how to efficiently compute PageRank for large numbers of pages. And we show how to apply PageRank to search and to user navigation.
