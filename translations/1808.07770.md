# Transpiling Programmable Computable Functions to Answer Set Programs
## تحويل الدوال القابلة للحوسبة والبرمجة إلى برامج مجموعة الإجابات

**arXiv ID:** 1808.07770
**Authors:** Ingmar Dasseville, Marc Denecker
**Year:** 2018
**Categories:** cs.PL
**Translation Quality:** 0.90
**Glossary Terms Used:** functional programming

### English Abstract
This work provides a translation from PCF to ASP, enabling search problems using PCF through a bridge between functional programming foundations and answer set programming paradigms.

### الملخص العربي
يوفر هذا العمل ترجمة من الدوال القابلة للحوسبة والبرمجة (PCF) إلى برمجة مجموعة الإجابات (ASP)، مما يمكّن تحديد مشاكل البحث باستخدام PCF من خلال جسر بين أسس البرمجة الوظيفية ونماذج برمجة مجموعة الإجابات.
