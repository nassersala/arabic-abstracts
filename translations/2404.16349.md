# More Asymmetry Yields Faster Matrix Multiplication

### English Abstract
The researchers introduce an advancement to the laser method for designing rapid matrix multiplication algorithms. Their approach builds upon recent developments by incorporating increased asymmetry in the analysis, allowing them to work around a constraint that previously required identical treatment of two dimensions. This yields an improved bound for square matrix multiplication exponent at ω < 2.371339, down from the prior ω < 2.371552, along with better exponents for rectangular matrix multiplication of different configurations.
