### English Abstract
This empirical study examines Transformer robustness under semantic-preserving code changes. The researchers implement 51 transformation strategies across Java and Python, categorized into five types: block, insertion/deletion, grammatical statement, grammatical token, and identifier modifications. Testing on code completion, summarization, and search reveals insertion/deletion and identifier changes pose greatest challenges, with abstract syntax tree-based models demonstrating superior robustness compared to sequence-only variants.
