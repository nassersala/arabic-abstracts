# Stream-HLS: Towards Automatic Dataflow Acceleration
## Stream-HLS: نحو التسريع التلقائي لتدفق البيانات

**arXiv ID:** 2501.09118
**Authors:** Suhail Basalama, Jason Cong
**Year:** 2025
**Categories:** Hardware Architecture (cs.AR)
**Translation Quality:** 0.93
**Glossary Terms Used:** optimization, performance, framework

### English Abstract
This paper addresses challenges in hardware circuit development using high-level synthesis. The authors propose Stream-HLS, a framework built on MLIR that takes a C/C++ or PyTorch software code and automatically generates an optimized dataflow architecture for FPGAs. Key innovations include an analytical performance model for dataflow optimization and support for multi-kernel applications with graph-level pipelining. The framework demonstrated significant performance improvements, achieving up to 79.43× and 10.62× geometric means better performance compared to prior automation and abstraction frameworks respectively.
