### English Abstract
The paper introduces a C++ language extension using attributes to help developers select optimal memory layouts between array-of-structs and structure-of-arrays formats. The compiler automatically converts data into the preferred arrangement based on execution context and algorithm requirements. The authors implemented this extension in Clang and validated it using smoothed particle hydrodynamics code tested on Intel CPUs, ARM CPUs, and Grace-Hopper GPUs.
