# Deriving monadic quicksort (Declarative Pearl)
## اشتقاق الترتيب السريع المونادي (لؤلؤة تصريحية)

**arXiv ID:** 2101.11421
**Authors:** Shin-Cheng Mu, Tsung-Ju Chiang
**Year:** 2021
**Categories:** cs.PL, cs.DS
**Translation Quality:** 0.90
**Glossary Terms Used:** monad, functional programming

### English Abstract
Demonstrates monadic program derivation by specifying sorting via the non-determinism monad, deriving both pure and state-monadic quicksort variants using functional techniques.

يوضح اشتقاق البرنامج المونادي من خلال تحديد الترتيب عبر موناد اللاحتمية، مع اشتقاق متغيرات الترتيب السريع الخالصة والمونادية الحالية باستخدام تقنيات وظيفية.
