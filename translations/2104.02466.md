# A Review of Formal Methods applied to Machine Learning

### English Abstract
This comprehensive review examines formal methods for verifying machine learning systems. It contrasts established formal methods in safety-critical domains like aviation with emerging techniques for neural networks, covering SMT, optimization, and abstract interpretation approaches. Discussion extends to support vector machines, decision trees, and often-neglected training/data preparation phases.
