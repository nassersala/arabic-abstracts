# Memory Hierarchy Sensitive Graph Layout

### English Abstract
The paper addresses how graph-structured data can be organized in memory to improve spatial locality across multiple levels of the memory hierarchy. Rather than optimizing for a single unit of locality, the author proposes hierarchical blocking—a technique that arranges nodes using successive breadth-first searches according to specified memory hierarchy parameters. The approach works with trees and general graphs, requiring only constant extra space.
