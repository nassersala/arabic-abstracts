# COLE: A Column-based Learned Storage for Blockchain Systems
## COLE: تخزين عمودي معتمد على التعلم لأنظمة البلوك تشين

**arXiv ID:** 2306.10739
**Authors:** Ce Zhang, Cheng Xu, Haibo Hu, Jianliang Xu
**Year:** 2024
**Categories:** Databases (cs.DB)
**Translation Quality:** 0.90
**Glossary Terms Used:** database, optimization, performance

### English Abstract
The paper addresses blockchain systems' substantial storage burden, noting that the storage cost mostly comes from the index, i.e., Merkle Patricia Trie (MPT). The researchers propose COLE, which employs column-based design principles to store historical state values indexed by learned models. This approach enables efficient data retrieval while supporting provenance queries. COLE achieves impressive improvements, reducing storage size by up to 94% while improving system throughput by 1.4×–5.4× compared to traditional MPT implementations.
