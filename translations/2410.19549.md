# Mirror Matrix on the Wall: coding and vector notation as tools for introspection

### English Abstract
The vector notation adopted by GNU Octave plays a significant role as an introspection tool, reflecting Kenneth E. Iverson's vision that programming languages should function as thinking instruments for problem representation and reasoning. This work explores vector notation in GNU Octave through analysis of operators and functions, seeking closer alignment with mathematical notation and improved code efficiency. The paper examines fundamental concepts including indexing, broadcasting, and function handles with case studies. By adopting vector notation, GNU Octave becomes a powerful resource for mathematicians, scientists, and engineers to articulate and address complex problems more effectively and intuitively.
