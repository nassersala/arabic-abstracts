# Authenticated Private Set Intersection: A Merkle Tree-Based Approach for Enhancing Data Integrity
## تقاطع المجموعة الخاص المصادق عليه: نهج قائم على شجرة ميركل لتعزيز سلامة البيانات

**arXiv ID:** 2506.04647

**Year:** 2025

### English Abstract
Private Set Intersection (PSI) enables secure computation of set intersections while preserving participant privacy, standard PSI existing protocols remain vulnerable to data integrity attacks allowing malicious participants to extract additional intersection information or mislead other parties. The authors propose defining data integrity in PSI and construct authenticated schemes by integrating Merkle Trees with volePSI and mPSI protocols. The two-party construction achieves communication complexity O(nλ+n log n), matching unauthenticated PSI performance, while the multi-party version reaches O(nκ+n log n) with additional overhead from Merkle tree inclusion proofs. The authenticated schemes incur higher computational costs than unauthenticated alternatives but provide integrity verification capabilities, with efficient implementations and potential improvements discussed.
