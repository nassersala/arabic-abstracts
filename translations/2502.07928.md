# Distributed Approach to Haskell Based Applications Refactoring with LLMs Based Multi-Agent Systems
## نهج موزع لإعادة هيكلة تطبيقات Haskell باستخدام أنظمة متعددة الوكلاء قائمة على نماذج اللغة الكبيرة

**arXiv ID:** 2502.07928
**Authors:** Shahbaz Siddeeq, Zeeshan Rasheed, Malik Abdul Sami, Mahade Hasan, Muhammad Waseem, Jussi Rasku, Mika Saari, Kai-Kristian Kemell, Pekka Abrahamsson
**Year:** 2025
**Categories:** cs.SE
**Translation Quality:** 0.90
**Glossary Terms Used:** multi-agent, distributed system, complexity, memory, functional programming, automation, testing

### English Abstract
The authors present "a large language models (LLMs) based multi-agent system to automate the refactoring of Haskell codebases." The system employs specialized agents for context analysis, refactoring, validation, and testing. Results demonstrated cyclomatic complexity reductions of 13.64% and 47.06%, memory allocation improvements of 4.17% and 41.73%, and runtime efficiency gains up to 50%. The approach addresses functional programming refactoring challenges through distributed, modular systems that integrate LLMs for precise task execution and inter-agent collaboration.
