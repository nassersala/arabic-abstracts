# Logical Relations for Formally Verified Authenticated Data Structures
## العلاقات المنطقية لبنى البيانات المصادق عليها والمتحقق منها رسمياً

**arXiv ID:** 2501.10802

**Year:** 2025

### English Abstract
The paper presents a formal security proof for a library that automatically generates authenticated versions of data structures. The approach employs a new relational separation logic for reasoning about programs that use collision-resistant cryptographic hash functions. The researchers construct two semantic models within a type system to justify how the library leverages type abstraction for enforcing security and correctness. They verify several optimization strategies and demonstrate how hand-written implementations can be safely integrated with automatically generated code. All findings have been mechanized in the Coq proof assistant using the Iris framework.
