# Stateful Dataflow Multigraphs: A Data-Centric Model for Performance Portability on Heterogeneous Architectures

### English Abstract
The paper introduces SDFGs (Stateful DataFlow multiGraphs), an intermediate representation designed to separate program design from optimization. The approach enables separating program definition from its optimization by merging detailed data dependencies with control-flow structures. The framework supports program transformations like tiling and double-buffering through graph rewriting and pattern matching. The authors demonstrate effectiveness across multiple hardware platforms and computational domains, showing scientists can develop code once and deploy it with strong performance across CPUs, GPUs, and FPGAs.
