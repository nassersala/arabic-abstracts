# From Logic to Functional Logic Programs
## من البرامج المنطقية إلى البرامج المنطقية الوظيفية

**arXiv ID:** 2205.06841
**Authors:** Michael Hanus
**Year:** 2022
**Categories:** cs.PL
**Translation Quality:** 0.92
**Glossary Terms Used:** logic programming, functional programming

### English Abstract
An automatic transformation maps logic programs into functional logic programs using functional dependency inference. This approach maintains the flexibility of logic programming but can improve computations by reducing infinite search spaces to finite ones.

يقدم تحويل تلقائي يربط البرامج المنطقية بالبرامج المنطقية الوظيفية باستخدام استنتاج الاعتماديات الوظيفية. يحافظ هذا النهج على مرونة البرمجة المنطقية لكن يمكنه تحسين العمليات الحسابية عن طريق تقليل فضاءات البحث اللانهائية إلى فضاءات محدودة.
