# A Purely Functional Computer Algebra System Embedded in Haskell
## نظام جبر حاسوبي وظيفي بحت مدمج في Haskell

**arXiv ID:** 1807.01456
**Authors:** Hiromi Ishii
**Year:** 2018
**Categories:** cs.SC, cs.PL
**Translation Quality:** 0.91
**Glossary Terms Used:** functional programming, domain-specific language, algorithm, composable

### English Abstract
The paper demonstrates "how methods in Functional Programming can be used to implement a computer algebra system." The authors present the computational-algebra package, which functions as a domain-specific language embedded in Haskell. They highlight that this library achieves "safety, composability, and correctness" through functional programming techniques. The work implements advanced Gröbner basis algorithms, specifically Faugère's F₄ and F₅ algorithms, designed with composability in mind.
