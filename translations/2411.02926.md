# Privacy-Preserving Graph-Based Machine Learning with Fully Homomorphic Encryption for Collaborative Anti-Money Laundering
## تعلم الآلة القائم على الرسوم البيانية مع حفظ الخصوصية باستخدام التشفير المتماثل الكامل لمكافحة غسل الأموال التعاونية

### English Abstract
The research addresses financial crime detection through encrypted collaborative systems. The authors developed privacy-preserving pipelines using TFHE technology integrated with Zama Concrete ML that enable institutions to share sensitive data securely. Two key contributions emerged: a Graph Neural Network pipeline and a graph-based XGBoost pipeline with a Graph Feature Preprocessor. Testing demonstrated the XGBoost model achieved over 99% accuracy, F1-score, precision, and recall on the balanced AML dataset in both unencrypted and FHE-encrypted inference settings. On imbalanced datasets, incorporating graph-based features improved the F1-score by 8%. The work emphasizes balancing privacy protections against computational demands.
