# Comprehensive Introduction to Fully Homomorphic Encryption for Dynamic Feedback Controller via LWE-based Cryptosystem
## مقدمة شاملة للتشفير المتماثل الكامل لمتحكم التغذية الراجعة الديناميكي عبر نظام تشفيري قائم على LWE

**arXiv ID:** 1904.08025
**Authors:** Junsoo Kim, Hyungbo Shim, Kyoohyung Han
**Year:** 2019
**Categories:** eess.SY, cs.CR
**Translation Quality:** 0.90
**Glossary Terms Used:** homomorphic encryption, cryptographic, quantum computing, security, controller

### English Abstract
The paper introduces an LWE-based cryptosystem, a post-quantum alternative that avoids dependency on factoring large primes. A key feature is support for fully homomorphic operations—two encrypted variables can be added and multiplied without decrypting them. However, error injection necessary for security prevents unlimited recursive operations. The authors demonstrate that dynamic feedback controllers, which ensure closed-loop stability, can overcome this limitation. The work includes MATLAB code examples designed specifically for control engineers unfamiliar with cryptography.
