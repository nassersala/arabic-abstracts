# Formal Computational Unlinkability Proofs of RFID Protocols
## براهين عدم الربط الحسابية الرسمية لبروتوكولات RFID

**arXiv ID:** 1705.02296
**Authors:** Hubert Comon, Adrien Koutsos
**Year:** 2017
**Categories:** cs.CR
**Translation Quality:** 0.90
**Glossary Terms Used:** None directly applicable

### English Abstract
Establishes a framework for formal proofs of RFID protocols using the computationally complete symbolic attacker model. Contributes axioms for hash function properties, formalizes computational unlinkability, and provides the first formal unlinkability proofs of RFID protocols in the computational model.

### الملخص العربي
يؤسس إطاراً للبراهين الرسمية لبروتوكولات RFID باستخدام نموذج المهاجم الرمزي الكامل حسابياً. يساهم ببديهيات لخصائص دوال التجزئة، ويصوغ رسمياً عدم الربط الحسابي، ويقدم أول براهين رسمية لعدم الربط في بروتوكولات RFID في النموذج الحسابي.
