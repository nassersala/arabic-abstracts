# LLM-based Multi-Agent System for Intelligent Refactoring of Haskell Code

### English Abstract
The paper proposes an LLM-based multi-agent system designed to automate code refactoring for Haskell programs. The system employs specialized agents handling code analysis, refactoring execution, verification, and debugging. Evaluation across open-source Haskell codebases demonstrated meaningful improvements: "average 11.03% decreased complexity in code, an improvement of 22.46% in overall code quality, and increase performance efficiency by an average of 13.27%." Additionally, memory usage was optimized up to 14.57%. The researchers conclude this approach shows promise for enhancing maintainability in functional programming language development through automation.
