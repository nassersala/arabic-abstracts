# Auditable data structures: theory and applications
## بنى البيانات القابلة للمراجعة: النظرية والتطبيقات

**arXiv ID:** 2306.01886
**Authors:** Andrea Canciani, Claudio Felicioli, Fabio Severino, Domenico Tortola
**Year:** 2023
**Categories:** Distributed, Parallel, and Cluster Computing (cs.DC); Cryptography and Security (cs.CR); Data Structures and Algorithms (cs.DS)

### English Abstract
The paper addresses how digital applications consuming external data can verify its trustworthiness. While authenticated data structures are able to authenticate the source when queried through generation of proofs, this work tackles a gap: enabling external auditors without prior data knowledge to verify historical consistency. The authors analyze the terminology and the current state of the art of the auditable data structures, then propose a general framework to support external audits from both internal and external users.
