### English Abstract
The paper introduces techniques for verifying data structure responses through cryptographic proofs, even when controlled by untrusted remote hosts. The authors present the path hash accumulator, a cryptographic hashing-based tool for validating properties of structured data represented as paths and decomposable queries on element sequences. Applications include graph connectivity queries, network path analysis, and geometric object operations like intersection detection and containment verification on two-dimensional structures.
