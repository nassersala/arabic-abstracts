# Application of property-based testing tools for metamorphic testing

### English Abstract
Metamorphic testing addresses challenges in software systems where conventional testing approaches are difficult to apply. This approach effectively tackles the test oracle problem and test case generation problem. The test oracle problem arises when determining correct expected outputs is difficult, or when actual outputs cannot be reliably compared to expected outcomes. Metamorphic relations provide formal specifications of systems under test. A key challenge involves generating effective test cases. Property-based testing generates test cases according to desired software properties, and can be viewed as a specialized form of property-based testing. This paper demonstrates how property-based testing tools can automate test generation and verification within metamorphic testing frameworks, enabling integration of both approaches under a unified testing methodology.
