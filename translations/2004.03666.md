# Compositional Formal Analysis Based on Conventional Engineering Models
## التحليل الرسمي التركيبي القائم على نماذج الهندسة التقليدية

**arXiv ID:** 2004.03666
**Authors:** Tyler D. Smith, Ryan Peroutka, Robert Edman
**Year:** 2020
**Categories:** cs.SE, cs.SY, eess.SY
**Translation Quality:** 0.91
**Glossary Terms Used:** simulation (محاكاة)

### English Abstract
Presents SLICED, a methodology reducing formal specification generation effort by deriving compositional state models from conventional engineering models. Demonstrated using NASA's Virtual ADAPT model and validated with Simulink simulation.

### الملخص العربي
تقدم SLICED، وهي منهجية تقلل جهد توليد المواصفات الرسمية من خلال اشتقاق نماذج الحالة التركيبية من نماذج الهندسة التقليدية. تم توضيحها باستخدام نموذج Virtual ADAPT التابع لناسا والتحقق منها باستخدام محاكاة Simulink.
