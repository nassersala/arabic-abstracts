# Pseudodeterministic Communication Complexity

### English Abstract
The paper demonstrates an n-bit partial function with randomized communication complexity O(log n) where completing it into a total function requires substantially higher complexity. This establishes an exponential separation between randomized and pseudodeterministic communication protocols, extending prior work on parity decision trees through lifting techniques.
