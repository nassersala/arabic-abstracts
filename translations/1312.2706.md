# Static Application-Level Race Detection in STM Haskell using Contracts
## اكتشاف سباق ثابت على مستوى التطبيق في STM Haskell باستخدام العقود

**arXiv ID:** 1312.2706
**Authors:** Romain Demeyer, Wim Vanhoof
**Year:** 2013
**Categories:** cs.LO, cs.DC, cs.PL
**Translation Quality:** 0.88
**Glossary Terms Used:** concurrent, function

### English Abstract
The paper addresses the challenge of writing concurrent programs using high-level synchronization primitives. The researchers propose "associate contracts to transactions" and define a program transformation enabling static contract checking for STM Haskell. Their approach allows developers to verify that transactions handle shared data while preserving consistency properties expressed as user-defined boolean functions, preventing harmful interference during concurrent execution.
