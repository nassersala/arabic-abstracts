# SIMD-ified R-tree Query Processing and Optimization
## معالجة استعلامات شجرة R المحسّنة بتقنية SIMD وتحسينها

**arXiv ID:** 2309.16913
**Authors:** Yeasir Rayhan, Walid G. Aref
**Year:** 2023
**Categories:** Databases (cs.DB)
**Translation Quality:** 0.92
**Glossary Terms Used:** database, optimization, performance

### English Abstract
This research explores leveraging Single Instruction Multiple Data (SIMD) vectorization in spatial database engines, specifically for R-tree indices. The authors present vectorized implementations of spatial range selection and join operations, testing two different R-tree node storage layouts. Their findings demonstrate that SIMD can improve the latency of the spatial query operators up to 9x.
