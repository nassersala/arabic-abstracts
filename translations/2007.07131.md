# Irregular Accesses Reorder Unit: Improving GPGPU Memory Coalescing for Graph-Based Workloads
## وحدة إعادة ترتيب الوصول غير المنتظم: تحسين دمج الذاكرة في GPGPU لأحمال العمل القائمة على الرسوم البيانية

**Year:** 2020
**Categories:** Hardware Architecture (cs.AR)

### English Abstract
The paper addresses performance challenges in GPU computing for irregular applications with unpredictable memory access patterns. The authors propose a hardware extension called the Irregular accesses Reorder Unit (IRU) that reorders data processed by the threads on irregular accesses which significantly improves memory coalescing. Key results include a 1.33x performance improvement and 13% energy savings with only 5.6% area overhead on graph-based applications.
