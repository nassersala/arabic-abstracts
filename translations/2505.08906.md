# Comparing Parallel Functional Array Languages: Programming and Performance

### English Abstract
Parallel functional array languages are an emerging class of programming languages that promise to combine low-effort parallel programming with good performance and performance portability. The study examines five languages—Accelerate, APL, DaCe, Futhark, and SaC—using four benchmarks: N-body simulation, MultiGrid, Quickhull, and Flash Attention. The researchers demonstrate that functional array code exhibits "much shorter and more comprehensible" characteristics compared to hand-optimized baselines. Testing involved 39 benchmark instances across a 32-core AMD EPYC system and NVIDIA A30 GPU. The authors conclude that "mature functional array languages have the potential to deliver performance competitive with the best available conventional techniques."
