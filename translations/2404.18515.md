# An Agile Formal Specification Language Design Based on K Framework
## تصميم لغة مواصفات رسمية رشيقة قائمة على إطار عمل K

**arXiv ID:** 2404.18515
**Authors:** Jianyu Zhang, Long Zhang, Yixuan Wu, Feng Yang
**Year:** 2024
**Categories:** cs.SE
**Translation Quality:** 0.91

### English Abstract
Introduces an Agile Specification Language (ASL) designed to simplify formal specification writing using K Framework and YAML. The approach reduces specification code size while maintaining verification capability, addressing challenges in learning complex formal specification languages.

تقدم لغة مواصفات رشيقة (ASL) مصممة لتبسيط كتابة المواصفات الرسمية باستخدام إطار عمل K و YAML. يقلل النهج من حجم شفرة المواصفات مع الحفاظ على قدرة التحقق، معالجاً تحديات تعلم لغات المواصفات الرسمية المعقدة.
