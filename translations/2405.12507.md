# Compiler support for semi-manual AoS-to-SoA conversions with data views

### English Abstract
The paper proposes a C++ language extension using attributes that enables developers to direct compiler-driven memory layout conversions. The extension allows automatic transformation between array-of-structures and structure-of-arrays formats, which have different performance characteristics depending on computational context. Developers can selectively apply these conversions to data subsets through annotations.
