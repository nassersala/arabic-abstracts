# Learning functional programs with function invention and reuse
## تعلم البرامج الوظيفية مع ابتكار الدوال وإعادة استخدامها

**Authors:** Andrei Diaconu

### English Abstract
This inductive functional programming research explores modular program generation where reuse is important (if not crucial) for a variety of problems. Two algorithms and type-based pruning techniques are presented to reduce program size through function reuse.
