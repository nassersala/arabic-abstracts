### English Abstract
Remora is a higher-order, rank-polymorphic array-processing programming language in the same general category as APL and J, designed for parallel hardware execution. The paper provides an example-driven introduction to the language and its computational model, originally created by Iverson for APL. It covers two language variants: Dynamic Remora with dynamic typing (similar to Scheme or Lisp) and Explicitly Typed Remora with a static, dependent type system that enables compile-time array shape capture. The article serves as an introduction to rank-polymorphic array-processing as a computational model generally, beyond Remora specifics. Type inference details remain unaddressed as ongoing research.
