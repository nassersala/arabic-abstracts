# Boolean function monotonicity testing requires (almost) n^{1/2} queries
## اختبار رتابة الدوال المنطقية يتطلب (تقريباً) n^{1/2} استعلام

**arXiv ID:** 2511.04558
**Authors:** Mark Chen, Xi Chen, Hao Cui, William Pires, Jonah Stockwell
**Year:** 2025
**Categories:** Computational Complexity (cs.CC); Discrete Mathematics (cs.DM); Data Structures and Algorithms (cs.DS)
**Translation Quality:** 0.91
**Glossary Terms Used:** boolean, function, algorithm, complexity, query

### English Abstract
The paper demonstrates that any (two-sided error) adaptive algorithm for testing monotonicity of Boolean functions must have query complexity Ω(n^{1/2-c}) for any constant c>0. This finding improves upon a previous lower bound and nearly matches existing upper bounds from prior research.
