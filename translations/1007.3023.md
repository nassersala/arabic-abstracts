# Purely Functional Structured Programming
## البرمجة المنظمة الوظيفية الصرفة

**arXiv ID:** 1007.3023
**Authors:** Steven Obua
**Year:** 2010
**Categories:** cs.PL
**Translation Quality:** 0.93
**Glossary Terms Used:** parallelism (implied), programming paradigm

### English Abstract
The proposal introduces LINEAR SCOPE to merge structured programming with purely functional programming. This enables mainstream programmers can reap the benefits of purely functional programming like easily exploitable parallelism while using familiar syntax without needing monad knowledge.

### الملخص العربي
يقدم الاقتراح النطاق الخطي (LINEAR SCOPE) لدمج البرمجة المنظمة مع البرمجة الوظيفية الصرفة. يمكّن هذا المبرمجين العاديين من جني فوائد البرمجة الوظيفية الصرفة مثل التوازي القابل للاستغلال بسهولة، مع استخدام بنية نحوية مألوفة دون الحاجة إلى معرفة المونادات.
