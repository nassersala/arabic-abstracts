# OASIS: Object-based Analytics Storage for Intelligent SQL Query Offloading in Scientific Tabular Workloads
## OASIS: تخزين تحليلي قائم على الكائنات لتفريغ استعلامات SQL الذكية في أحمال العمل الجدولية العلمية

### English Abstract
The paper introduces a Computation-Enabled Object Storage system addressing limitations in existing solutions. OASIS achieves up to a 32.7% performance improvement over Spark configured with existing COS-based storage systems. Key contributions include flexible output formats supporting columnar layouts like Arrow, broad operator support including aggregation and sorting, and intelligent execution path selection across storage hierarchies optimized for scientific data analysis in HPC environments.
