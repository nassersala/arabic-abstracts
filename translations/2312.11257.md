# Destination-passing style programming: a Haskell implementation
## برمجة نمط تمرير الوجهة: تطبيق Haskell

**arXiv ID:** 2312.11257
**Authors:** Thomas Bagrel
**Year:** 2023
**Categories:** cs.PL
**Translation Quality:** 0.88
**Glossary Terms Used:** memory, function, parameter, implementation, performance

### English Abstract
The paper introduces a programming technique using "destinations, which represent the address of a write-once memory cell." These can be passed as function parameters, allowing callers to control memory management. The author demonstrates a Haskell implementation utilizing compact regions with linear typing to prevent uninitialized reads and memory errors while maintaining garbage collection benefits. A parser example shows the destination-based approach achieves "35% less memory and time than its naive counterpart for large inputs."
