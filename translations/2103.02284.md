# Columnar Storage and List-based Processing for Graph Database Management Systems
## التخزين العمودي والمعالجة القائمة على القوائم لأنظمة إدارة قواعد بيانات الرسوم البيانية

**arXiv ID:** 2103.02284
**Authors:** Pranjal Gupta, Amine Mhedhbi, Semih Salihoglu
**Year:** 2021
**Categories:** Databases (cs.DB)
**Translation Quality:** 0.89
**Glossary Terms Used:** database, data structure, optimization

### English Abstract
The paper revisits column-oriented storage within graph database systems. The researchers identified specific data access patterns unique to GDBMSs and developed optimized techniques including a novel list-based query processor, which avoids expensive data copies of traditional block-based processors under many-to-many joins. Additional innovations encompass new indexing structures for edge properties and compression methods using bit vectors.
