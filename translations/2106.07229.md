# Privacy-Preserving Machine Learning with Fully Homomorphic Encryption for Deep Neural Network

### English Abstract
The researchers implemented a ResNet-20 model using RNS-CKKS fully homomorphic encryption with bootstrapping capabilities. Their key innovation involves using state-of-the-art approximation methods to evaluate non-arithmetic functions like ReLU, rather than replacing them with simpler alternatives. This approach enabled deeper neural network evaluation on encrypted data. The model achieved 98.67% identical results to the original ResNet-20 model on CIFAR-10, with a classification accuracy of 90.67%—a substantial improvement over the previous maximum of 77% for FHE-based privacy-preserving machine learning systems.
