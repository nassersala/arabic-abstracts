### English Abstract
The paper presents convolution as a unified algebraic framework using semirings and semimodules. Key contributions include generalizing "Brzozowski's method for regular expression matching" by removing syntactic dependencies and extending it to weighted membership and n-ary relations. The work demonstrates how classical data structures like tries provide efficient alternatives, while showing that polynomial arithmetic and image convolution emerge as natural special cases within this framework.
