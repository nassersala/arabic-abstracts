# A Survey of Pipeline Tools for Data Engineering

### English Abstract
This research examines various pipeline tools used in data engineering workflows. The authors categorize these tools into four main groups: ETL/ELT systems, data integration and transformation pipelines, orchestration and workflow management solutions, and machine learning pipelines. The survey evaluates both commercial and open-source options, demonstrating through case studies how practitioners apply these tools to prepare datasets for machine learning applications while addressing common data wrangling challenges.
