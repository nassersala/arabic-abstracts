# Evaluation strategies for monadic computations
## استراتيجيات التقييم للحسابات الموناد

**arXiv ID:** 1202.2921
**Authors:** Tomas Petricek
**Year:** 2012
**Categories:** cs.PL
**Translation Quality:** 0.87
**Glossary Terms Used:** monad, functional programming, category

### English Abstract
The paper addresses how monads structure effectful computations in functional programming. The author proposes translating pure code to monadic form using an operation called "malias" that abstracts evaluation strategy. Key contributions include: implementations of malias providing "call-by-value and call-by-name semantics", a categorical framework specifying required operational laws, partial resolution of an open question regarding "call-by-need semantics", and a unified translation allowing "evaluation strategy of functional code translated to the monadic form" to be modified without altering structure or types.
