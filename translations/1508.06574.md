### English Abstract
The paper examines cryptographic advances enabling secure computation on encrypted data. It provides an accessible overview of homomorphic encryption schemes tailored for statisticians and machine learning practitioners, highlighting current limitations. The authors survey which statistical and machine learning algorithms can be practically implemented within these constraints, and document an R package implementing a recent homomorphic scheme.
