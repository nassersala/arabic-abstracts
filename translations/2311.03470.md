# Orion: A Fully Homomorphic Encryption Framework for Deep Learning
## Orion: إطار عمل للتشفير المتماثل الكامل للتعلم العميق

**arXiv ID:** 2311.03470
**Authors:** Austin Ebel, Karthik Garimella, Brandon Reagen
**Year:** 2023
**Categories:** cs.CR
**Translation Quality:** 0.94
**Glossary Terms Used:** framework, homomorphic encryption, bootstrapping, optimization, performance

### English Abstract
The paper introduces Orion, an automated framework enabling computation directly on encrypted data through fully homomorphic encryption applied to deep learning. The researchers developed novel techniques including a multiplexed packing strategy for convolutions and automated bootstrap management to address key FHE challenges. Key achievements include outperforming prior work by 2.38x on ResNet-20, and demonstrating capability on larger networks including ResNet-50 on ImageNet and YOLO-v1 object detection with 139 million parameters—described as the first high-resolution FHE object detection experiments. The framework accepts PyTorch models and has been released as open-source software.
