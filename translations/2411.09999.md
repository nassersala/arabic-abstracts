# Understanding Graph Databases: A Comprehensive Tutorial and Survey
## فهم قواعد بيانات الرسوم البيانية: دليل ومسح شامل

**arXiv ID:** 2411.09999
**Authors:** Sydney Anuyah, Victor Bolade, Oluwatosin Agbaakin
**Year:** 2024
**Categories:** Databases (cs.DB); Data Structures and Algorithms (cs.DS)
**Translation Quality:** 0.90
**Glossary Terms Used:** algorithm (خوارزمية), graph (رسم بياني), database (قاعدة بيانات), data structure (بنية البيانات), network (شبكة)

### English Abstract
This tutorial serves as a comprehensive guide for understanding graph databases, focusing on the fundamentals of graph theory while showcasing practical applications across various fields. It starts by introducing foundational concepts and delves into the structure of graphs through nodes and edges, covering different types such as undirected, directed, weighted, and unweighted graphs. Key graph properties, terminologies, and essential algorithms for network analysis are outlined, including Dijkstra's shortest path algorithm and methods for calculating node centrality and graph connectivity.

### الملخص العربي
يعمل هذا الدليل كدليل شامل لفهم قواعد بيانات الرسوم البيانية، مع التركيز على أساسيات نظرية الرسوم البيانية مع عرض التطبيقات العملية عبر مجالات مختلفة. يبدأ بتقديم المفاهيم الأساسية ويتعمق في بنية الرسوم البيانية من خلال العقد والحواف، ويغطي أنواعاً مختلفة مثل الرسوم البيانية غير الموجهة والموجهة والمرجحة وغير المرجحة. يتم تحديد خصائص الرسم البياني الرئيسية، والمصطلحات، والخوارزميات الأساسية لتحليل الشبكات، بما في ذلك خوارزمية ديكسترا لأقصر مسار وطرق حساب مركزية العقدة وترابط الرسم البياني.
