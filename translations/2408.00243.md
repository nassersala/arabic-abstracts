# A Survey on the Applications of Zero-Knowledge Proofs
## مسح حول تطبيقات براهين المعرفة الصفرية

**arXiv ID:** 2408.00243
**Authors:** Ryan Lavin, Xuekai Liu, Hardhik Mohanty, Logan Norman, Giovanni Zaarour, Bhaskar Krishnamachari
**Year:** 2024
**Categories:** Cryptography and Security (cs.CR); Computational Complexity (cs.CC)
**Translation Quality:** 0.90
**Glossary Terms Used:** zero-knowledge proof, cryptography, blockchain, privacy, security, authentication, machine learning, virtual machine, domain-specific language, framework

### English Abstract
The paper examines how zero-knowledge proofs enable secure and private exchange of information without revealing underlying private data. The survey focuses on zk-SNARKS and covers practical applications across multiple domains. These include blockchain uses (privacy, scaling, storage, interoperability) and non-blockchain applications like voting, authentication, and machine learning. The work discusses supporting infrastructure including zero-knowledge virtual machines, domain-specific languages, and related frameworks, concluding with future research directions.
