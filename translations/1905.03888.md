### English Abstract
The paper introduces Charlotte, a system for building composable, authenticated distributed data structures. Data blocks reference each other by hash, forming a "blockweb" directed acyclic graph. The framework enables applications to operate fully independently when possible, and share blocks when desired. Charlotte supports heterogeneous trust models where different observers maintain distinct beliefs about system failures. Key contributions include mechanisms for ensuring data immutability, self-authentication, and implementations demonstrating consensus and proof-of-work approaches. The authors showcase flexibility through various applications, suggesting the framework enables developers to build flexible, fast, composable applications with strong guarantees.
