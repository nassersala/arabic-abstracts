# DynaSOAr: Accelerating Single-Method Multiple-Objects Applications on GPUs
## DynaSOAr: تسريع تطبيقات الأسلوب الواحد-الكائنات المتعددة على وحدات معالجة الرسوميات

### English Abstract
The paper addresses the perception that object-oriented programming is inefficient for SIMD high-performance computing. Springer identifies a subset called Single-Method Multiple-Objects (SMMO) that can be implemented efficiently on massively parallel SIMD accelerators by executing methods across multiple objects simultaneously. The framework DynaSOAr, built for CUDA, enhances memory efficiency through an SOA (Structure of Arrays) data layout and achieves low memory fragmentation through efficient management of free and allocated memory blocks with lock-free, hierarchical bitmaps.
