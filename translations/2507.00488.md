# Have Object-Oriented Languages Missed a Trick with Class Function and its Subclasses?

### English Abstract
The paper investigates whether object-oriented languages lack formal "class Function" structures and specialized subclasses for functions with specific properties. It proposes implementing a class Function with subclasses, experimentally demonstrating these concepts across popular programming languages.
