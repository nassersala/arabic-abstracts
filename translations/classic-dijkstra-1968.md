# Go To Statement Considered Harmful
## جملة Go To تُعتبر ضارة

**Publication:** Communications of the ACM, 11(3):147-148, March 1968
**Authors:** Edsger Dijkstra
**Year:** 1968
**Domain:** Software Engineering
**Translation Quality:** 0.91
**Glossary Terms Used:** statement, programming language, algorithm, verification, correctness, structured programming, control flow, loop

### English Summary
This letter to the editor argued that unrestricted GOTO statements should be abolished from higher-level languages because they complicated the task of analyzing and verifying the correctness of programs (particularly those involving loops). To many people, this letter marks the true beginning of structured programming. The original title submitted was "A Case Against the Goto Statement", but the CACM editor changed it to "Goto Statement Considered Harmful", spawning the widely-used "considered harmful" phrase template.
