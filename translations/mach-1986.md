### English Abstract
Mach is an operating system kernel developed at Carnegie Mellon University to support operating system research and development. Mach has three major design goals: support a single system image across networks of uniprocessors and multiprocessors; provide object-oriented APIs with abstractions for task management, scheduling, inter-process communication, memory objects, and device management; provide flexible, efficient support for a variety of parallel and distributed computing paradigms. The Mach kernel provides these capabilities through a small set of basic abstractions: tasks, threads, ports, messages, and memory objects. Tasks are execution environments with protected address spaces. Threads are the basic unit of CPU utilization in Mach. Ports are kernel-protected communication channels that serve as the reference points for kernel-mediated IPC. Messages are typed collections of data objects transmitted between threads via ports. Memory objects are internal units of memory management that can be mapped into task address spaces.

### Notes
Mach is one of the most influential operating system kernels in computer science history. It introduced the microkernel architecture that influenced macOS, iOS, and GNU Hurd. The kernel's message-passing IPC mechanism and separation of policy from mechanism became foundational concepts in OS design.
