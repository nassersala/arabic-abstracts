# Deterministic list decoding of Reed-Solomon codes
## فك الترميز الحتمي بالقائمة لرموز ريد-سولومون

**arXiv ID:** 2511.05176
**Authors:** Soham Chatterjee, Prahladh Harsha, Mrinal Kumar
**Year:** 2025
**Categories:** Computational Complexity (cs.CC); Information Theory (cs.IT)
**Translation Quality:** 0.92
**Glossary Terms Used:** deterministic, algorithm, polynomial, complexity, efficiency

### English Abstract
The researchers demonstrate that Reed-Solomon codes of dimension k and block length n over any finite field F can be deterministically list decoded from agreement √((k-1)n) in time poly(n, log |F|). This advances prior work by Sudan and Guruswami-Sudan, which required either randomization or time complexity depending on field characteristics. The key innovation involves developing a deterministic algorithm for bivariate polynomial factorization instances that arise during list decoding, achieving efficiency independent of field size in ways previously thought difficult.
