# Haskelite: A Tracing Interpreter Based on a Pattern-Matching Calculus
## Haskelite: مفسر تتبعي قائم على حساب مطابقة الأنماط

**arXiv ID:** 2407.11831
**Authors:** Pedro Vasconcelos, Rodrigo Marques
**Year:** 2024
**Categories:** cs.PL
**Translation Quality:** 0.90
**Glossary Terms Used:** functional programming, pattern matching, semantic, implementation

### English Abstract
The paper addresses how functional program evaluation is typically explained in textbooks through stepwise rewriting, but standard implementation techniques complicate producing corresponding traces. The authors present "a tracing interpreter for a subset of Haskell based on the pattern matching calculus of Kahl." They develop semantics progressing from big-step to small-step styles, implement this in an educational interpreter, and discuss implementation choices with examples.
