# A Layered Implementation Framework for Regular Languages
## إطار تطبيق مُطَبَّق متعدد الطبقات للغات النظامية

**arXiv ID:** 2509.18232
**Authors:** Baudouin Le Charlier
**Year:** 2025
**Categories:** cs.FL (Formal Languages and Automata Theory)
**Translation Quality:** 0.88
**Glossary Terms Used:** implementation, regular expression, syntax, derivative, deterministic, equivalent, framework

### English Abstract
We present an implemented system designed to manipulate representations of regular languages, structured in two layers. The first layer introduces normalized regular expressions with finite syntactic derivatives, represented by unique integer identifiers at the implementation level. The second layer, called the background, unifies equivalent expressions into equivalence classes with representative shortest expressions and derivative equations. The work demonstrates how this framework enables unified representation of large sets of regular languages, where distinct identifiers correspond to different languages, each with both a small expression and a minimal deterministic automaton.
